# 动漫数字人AI语音助手

## 项目简介
动漫数字人AI语音助手是一个基于DeepSeek AI的互动助手，旨在提供友好的用户体验。用户可以通过文本或语音与助手进行对话，助手将根据用户的输入提供智能回复。

## 文件结构
```
anime-ai-assistant
├── src
│   ├── 1.html          # 应用的主HTML文件，包含页面的结构和内容
│   ├── css
│   │   └── styles.css  # 应用的样式定义
│   ├── js
│   │   └── main.js     # 应用的主JavaScript文件，负责交互逻辑
│   └── libs
│       └── deepseek-client.js  # DeepSeek API的客户端库
├── .vscode
│   └── settings.json   # VS Code的工作区设置
├── package.json         # npm的配置文件，列出项目的依赖项和脚本
├── .gitignore           # 指定在版本控制中应忽略的文件和文件夹
└── README.md            # 项目的文档
```

## 使用方法
1. 克隆项目到本地：
   ```
   git clone <repository-url>
   ```
2. 进入项目目录：
   ```
   cd anime-ai-assistant
   ```
3. 安装依赖：
   ```
   npm install
   ```
4. 启动应用：
   ```
   npm start
   ```

## 功能
- 文本和语音输入
- 实时对话
- 语音合成回复
- 友好的用户界面
